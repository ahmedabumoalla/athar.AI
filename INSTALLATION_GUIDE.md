# دليل تثبيت وتشغيل منصة أثر AI

أثر AI هي منصة إرشاد أكاديمي ذكية تساعد طلاب الجامعات على فهم مستواهم الأكاديمي، تحليل نقاط القوة والضعف، الحصول على توصيات مخصصة، وتحسين رحلتهم الجامعية باستخدام الذكاء الاصطناعي.

تهدف المنصة إلى تحويل الإرشاد الأكاديمي من نصائح عامة إلى تجربة ذكية وشخصية تعتمد على بيانات الطالب الفعلية، مثل التخصص، المعدل، المواد الصعبة، نقاط القوة، أسلوب التعلم، الأهداف الأكاديمية، والتقدم داخل المنصة.

المنصة تحتوي على الصفحة الرئيسية، تسجيل الدخول، إنشاء حساب طالب جديد، لوحة تحكم الطالب، المساعد الأكاديمي الذكي، الدورات المساعدة، الأنشطة التدريبية، التدقيق الذكي للمستوى، تحليل ملاءمة التخصص، واقتراح مواد الترم القادم.

---

# التقنيات المستخدمة في المشروع

تم بناء المشروع باستخدام Next.js 16 بنظام App Router، وهو الإطار الأساسي المستخدم لبناء صفحات المنصة، إدارة المسارات، إنشاء API Routes، تشغيل منطق الخادم، وتجهيز المشروع للنشر على Vercel.

تم استخدام React 19 لبناء واجهات المستخدم التفاعلية، وإدارة حالة الصفحات، والتحكم بالنماذج، وعرض بيانات الطالب القادمة من Supabase، وعرض نتائج الذكاء الاصطناعي داخل الصفحات.

تم استخدام TypeScript لتنظيم الكود وتقليل الأخطاء، وتعريف أنواع البيانات مثل بيانات الطالب، التحليلات الأكاديمية، الأنشطة، نتائج الذكاء الاصطناعي، المواد المقترحة، ورسائل المساعد الذكي.

تم استخدام Tailwind CSS لتصميم واجهات المنصة بالكامل، بما يشمل الخلفيات، البطاقات، الأزرار، النماذج، الظلال، الحواف الدائرية، الواجهة العربية RTL، والتصميم الأكاديمي الهادئ المعتمد على خلفية فاتحة وبطاقات بيضاء شفافة وهوية زرقاء مريحة.

تم استخدام Supabase كنظام خلفي للمصادقة وقاعدة البيانات. يستخدم Supabase في تسجيل الدخول، إنشاء الحسابات، إدارة جلسات المستخدمين، حفظ بيانات الطالب، حفظ التحليلات الأكاديمية، حفظ الأنشطة التدريبية، تتبع الدورات المكتملة، وجلب البيانات المرتبطة بالمستخدم الحالي.

تم استخدام OpenAI API لتشغيل التحليل الذكي داخل المنصة. يستخدم OpenAI في المساعد الأكاديمي، توليد الأنشطة التدريبية، تحليل مستوى الطالب، تحليل ملاءمة التخصص، اقتراح مواد الترم القادم، وتوليد التوصيات الأكاديمية المخصصة.

تم استخدام مكتبة Lucide React للأيقونات داخل الواجهة، مثل أيقونات الرجوع، التحميل، التحليل، الهدف، الدورات، التقويم، ونقاط التقدم.

تم استخدام Vercel لنشر المشروع واستضافة تطبيق Next.js وتشغيل صفحات الواجهة و API Routes من خلال بيئة إنتاجية.

# متطلبات التشغيل

قبل تشغيل المشروع يجب توفر Node.js و npm، بالإضافة إلى مشروع Supabase جاهز، ومفتاح OpenAI API، وملف البيئة `.env.local`.

يجب إنشاء ملف باسم `.env.local` داخل جذر المشروع وإضافة القيم التالية:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
OPENAI_API_KEY=your_openai_api_key
